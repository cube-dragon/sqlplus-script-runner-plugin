Definir o parâmetro do ambiente NLS_LANG é a maneira mais simples de especificar o comportamento da localidade para o software Oracle. Ele define o idioma e o território usados pelo aplicativo cliente e pelo servidor de banco de dados, além de especificar convenções como a data padrão, os formatos monetário e numérico. Exemplo: AMERICAN_AMERICA.US7ASCII ou FRENCH_CANADA.WE8ISO8859P1 Fonte: ORAFAQ
